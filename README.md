# FEMA Disaster Intelligence Platform

An interactive data analytics dashboard built using **Streamlit** and **Plotly** to analyze over 70 years of FEMA disaster declarations (1953–2025). The platform provides insights into disaster trends, geographic patterns, and future risk predictions.

---

## 🚀 Features
- 📈 Temporal trend analysis (yearly trends, growth rates, seasonality)
- 🗺️ Geographic visualization (state-level and regional insights)
- 🔬 Incident type analysis and distribution
- ⚠️ Risk scoring to identify high-risk states
- 🔮 Forecasting of future disaster trends
- 📊 Interactive filters for dynamic exploration

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- Plotly  
- NumPy  
- Scikit-learn
